# Task Manager API

A simple beginner-friendly backend project built with Node.js and Express.  
This API allows users to create, read, update, delete, and filter tasks.

## Features

- Get all tasks
- Get a single task by ID
- Create a new task
- Update an existing task
- Delete a task
- Filter tasks by completion status
- Basic request validation

## Tech Stack

- Node.js
- Express.js
